# Sway Window Manager Configuration

## Contents
  - `config`: Main configuration file
  - `swaybar.sh`: Command-line sway-bar script (incomplete)

## Dependencies
The configuration file depends on these programs for intended usage:
  - `swayidle`
  - `swaylock`
  - `clipman`
  - `wl-paste`
  - `alacritty`
  - `firefox`
  - `wofi`
  - `pactl`
  - `light`
  - `acpi`
